# MyGrind

## Project Overview
MyGrind is a fitness and habit tracking app designed to help users build and maintain healthy habits while achieving their fitness goals. The app provides an intuitive interface to monitor progress, set achievable targets, and stay motivated.

## Features
- **User-friendly Dashboard**: View all your fitness and habit data at a glance.
- **Habit Tracking**: Easily track daily habits with reminders and progress visuals.
- **Goal Setting**: Set and manage fitness goals with personalized plans.
- **Progress Analytics**: Analyze your habits and fitness performance over time.
- **Community Support**: Connect with friends and share progress for motivation.

## Setup Instructions
### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)
- MongoDB (for backend storage)

### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/124110492-ui/my-grind.git
   cd my-grind
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Set up the environment variables:
   - Create a `.env` file in the root directory and configure your MongoDB URI.

4. Run the application:
   ```bash
   npm start
   ```

### Usage
- Access the app in your browser at `http://localhost:3000`.
